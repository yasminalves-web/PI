# Pinhalense auxílio de pesquisa e automatização – Estudo de Caso

 ## RF01 — Recomendação de Layouts com base no Croqui enviado
 O sistema receberá um Croqui e o analisará e comparará aos Layouts já existentes e recomendará o mais próximo ao usuário.
 
## RF02 — Importação de dados

 O sistema realizara a importação dos dados do iLogic do Inventor para o Excel com base nas especificações do usuário.

 ## RF03 — Exportação de dados

 O sistema exportará os dados validados para um arquivo Excel (.xlsx).
